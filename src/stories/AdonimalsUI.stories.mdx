# Welcome to AdonimalsUI

O objetivo desse design system é ser, inicialmente, um projeto de estudos, tanto de React quanto de Typescript, Styled Systyem e Styled Components.

Por conta disso, o projeto é simples e irá crescer de acordo com as necessidades de outro projeto, a saber, Adonimals, que é uma rede social para adoção de animals.
